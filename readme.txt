Petit script de traduction en ligne de commande

Utilise l'API IBM Watson Language Translator
Necéssite une apikey obtenable sur https://www.ibm.com/watson/services/language-translator/

Utilisé sans option le script traduit par défaut de l'anglais vers le français
-e --english => traduit vers l'anglais
-f --francais => traduit vers le francais
-g --graphik => lance l'interface graphique
